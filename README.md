# Real Time Posture Analysis For Industrial Weightlifting

In this modern era where people fear the rise and hope of AGI there still are industries where man power is used to lift heavy objects and lifting heavy things comes with a pre-requisite of correct posture and neat technique. If not done with an efficient posture the action of lifting heavy weights could incurr a lifelong injury to the person.

Hence in this repo we introduce a application which detects such incorrect posture and aims to provide real time updates and notifications to people to prevent any injuries and damages.
